## 🔊 What is sonar-vice-widget-macos?

sonar-vice-widget-macos is a simple desktop widget for macOS. It stays on top of other windows, letting you control SteelSeries Sonar audio settings, equalizer profiles, and RGB lighting without opening the full SteelSeries GG software. The widget floats on your screen, always at hand when you want to change sound or lighting quickly.

This tool helps users who want faster access to key SteelSeries features while gaming or working. It fits nicely on the desktop and stays visible, so you don't lose time switching windows.

## ❓ Troubleshooting Common Issues

### The Widget Won't Open

- Make sure you moved it to the Applications folder before running.  
- Check if macOS blocked the app in **Security & Privacy** and allow it there.  
- Restart your Mac and try again.

### The Widget Does Not Control Sonar or RGB

- Confirm SteelSeries GG is running with your devices connected.  
- Make sure the widget has permission in macOS Privacy settings.  
- If you updated SteelSeries software recently, restart the widget.

### Widget Window Disappears

- Try dragging your mouse to the edges of your desktop.  
- It may be hidden behind other windows.  
- Use Mission Control on macOS (swipe up) to find the widget.

## 📋 Frequently Asked Questions

**Q: Do I need to keep SteelSeries GG open?**  
Yes. sonar-vice-widget-macos works as a companion tool and requires SteelSeries GG and Sonar to be running.

**Q: Can I use this on Windows?**  
No. This widget was built specifically for macOS and does not run on Windows.

**Q: Does it support all SteelSeries devices?**  
It supports devices compatible with the Sonar software's audio and RGB controls, but some older models may not work fully.
